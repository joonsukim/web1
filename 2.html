CSS

A style sheet format for HTML documents endorsed by the World Wide Web Consortium. CSS1 (Version 1.0) provided hundreds of layout settings that can be applied to all the subsequent HTML pages that are downloaded. CSS2 (Version 2.0) added support for XML, oral presentations for the visually impaired, downloadable fonts and other enhancements.Comprising some 50 modules, CSS3 (Version 3.0) added features such as vertical text, elaborate borders and backgrounds, user interaction and greater device and browser detection. For information, visit www.w3.org/Style/CSS/. See HTML, style sheet and XSL.
